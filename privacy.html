Resumen en una línea

teologia.media no recopila, almacena ni procesa datos personales de visitantes. La aplicación asociada a este sitio usa la API de YouTube únicamente para publicar y administrar nuestros propios canales.

Qué sí hace la aplicación de Google OAuth asociada

teologia.media opera una aplicación interna de Google Cloud que solicita, bajo consentimiento del propietario de los canales (Philipp Muesch y Fernanda - los operadores), los siguientes permisos de YouTube Data API v3:

  • youtube.upload - subir videos a nuestro propio canal (Camino de Dios).
  • youtube - administrar metadatos (títulos, descripciones, etiquetas, estado público/oculto) de nuestros propios canales.

Estos permisos se ejercen exclusivamente sobre los canales de YouTube propiedad de los operadores del sitio. La aplicación no accede, no lee, no modifica ni almacena datos de ningún otro canal, cuenta de Google, ni usuario.

Qué no hace la aplicación

  • No recopila datos personales de visitantes al sitio.
  • No coloca cookies de seguimiento ni balizas web de terceros.
  • No ejecuta JavaScript de análisis (Google Analytics, Meta Pixel, etc.).
  • No vende, comparte ni transfiere datos a terceros - porque no los recopila.
  • No lee los comentarios, suscripciones, ni datos de otras cuentas de YouTube.

Infraestructura

El sitio se aloja en Cloudflare Pages. Cloudflare puede recopilar registros técnicos agregados (dirección IP, agente de usuario) con fines de seguridad y prevención de abuso, conforme a su propia política de privacidad. teologia.media no accede a esos registros individuales.

Contacto

Para consultas sobre esta política o sobre el tratamiento de datos, escribe a contacto@teologia.studio.

Ley aplicable

Esta política se rige por el derecho suizo. Cualquier disputa será resuelta por los tribunales competentes del cantón de Zúrich, Suiza.

One-line summary

teologia.media does not collect, store, or process personal data from site visitors. The Google OAuth application associated with this site uses the YouTube API solely to publish and manage our own channels.

What the Google OAuth application does

teologia.media operates an internal Google Cloud application that requests - with the consent of the channel owners (Philipp Muesch and Fernanda - the site operators) - the following YouTube Data API v3 scopes:

  • youtube.upload - to upload videos to our own channel (Camino de Dios).
  • youtube - to manage metadata (titles, descriptions, tags, public/unlisted status) on our own channels.

These scopes are exercised exclusively on the YouTube channels owned by the site operators. The application does not access, read, modify, or store data from any other channel, Google account, or user.

What the application does not do

  • It does not collect personal data from site visitors.
  • It does not place third-party tracking cookies or web beacons.
  • It does not run analytics JavaScript (Google Analytics, Meta Pixel, etc.).
  • It does not sell, share, or transfer data to third parties - because it does not collect any.
  • It does not read comments, subscriptions, or data from other YouTube accounts.

Infrastructure

The site is hosted on Cloudflare Pages. Cloudflare may collect aggregated technical logs (IP address, user agent) for security and abuse-prevention purposes, in accordance with Cloudflare's own privacy policy. teologia.media does not access those individual logs.

Contact

For questions about this policy or about data handling, please write to contacto@teologia.studio.
